# ExoHabitAI

## Project Description
ExoHabitAI is an AI-based system that predicts the habitability potential of 
exoplanets using planetary and stellar parameters.

## Tech Stack
- Python
- Machine Learning
- Flask
- HTML, CSS, JavaScript
- Data Visualization
